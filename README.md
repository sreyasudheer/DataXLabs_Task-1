# Task 1: Data Cleaning and Preprocessing

## Objective
To clean and prepare a raw Sales Dataset using Microsoft Excel by handling missing values, removing duplicate entries, and standardizing formatting to make it industry-ready.

## Repository Structure
The uploaded Excel files contains three organized sheets:
1. **Original:** The raw unedited dataset downloaded from Kaggle.
2. **Cleaned copy:** The final processed data after treatment.
3. **Cleaned summary:** A brief overview of the changes made.

## Actions Taken in Excel
- **Handled Duplicate Rows:** Used Excel's "Remove Duplicates" tool under the Data tab to eliminate redundant transaction records.
- **Fixed Missing Values:** Used filters to identify blank cells in key columns and treated them carefully.
- **Date Standardization:** Formatted the entire 'Sale_Date' column into a uniform date format.
- **Text & Column Cleaning:** Trimmed extra spaces and cleaned the column headers to be uniform and easy to read.
